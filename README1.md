Exploratory Data Analysis (EDA) Project

Project Overview
This project performs Exploratory Data Analysis (EDA) on a Food Delivery dataset to identify patterns, trends, correlations, and key influencing factors affecting food delivery services.

The analysis uses statistical summaries and visualizations to gain meaningful insights from the dataset.

Objectives
- Analyze and explore dataset structure
- Identify patterns and trends
- Perform statistical analysis
- Visualize relationships between variables
- Generate insights from the data

Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn

Dataset Information
The dataset contains information related to food delivery services including:

- Restaurant Type
- Delivery Time
- Order Value
- Customer Rating
- Distance (km)
- Weather Conditions

Analysis Performed

Data Exploration
- Dataset structure analysis
- Shape and information of dataset
- Missing value detection
- Duplicate value checking

Statistical Summary
Used statistical measures such as:
- Mean
- Minimum & Maximum values
- Standard Deviation
- Quartiles

Visualizations
The following visualizations were created:
- Restaurant Type Distribution
- Average Order Value by Restaurant Type
- Delivery Time by Weather
- Customer Rating Distribution
- Distance vs Delivery Time Scatter Plot
- Correlation Heatmap
- Outlier Detection

Correlation Analysis
Correlation analysis was performed to identify relationships between numerical variables.

Key Insights
- Italian restaurants have the highest average order value.
- Rainy weather increases delivery time significantly.
- Longer delivery distances result in higher delivery times.
- Most customer ratings are above 4.0.
- Fast Food orders are the most common.
